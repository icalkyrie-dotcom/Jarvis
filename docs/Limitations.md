# Current Limitations

The project prioritizes reliability and maintainability over feature completeness.

---

## Current Limitations

- Markdown knowledge files only
- Android client only
- Tavily Free search limitations
- No background scheduler
- No autonomous agents yet

---

## Planned Improvements

- PDF support
- DOCX support
- Better embedding models
- Background summarization
- Agent workflows
- Desktop application
- Multi-device synchronization

---

These limitations are tracked through the project roadmap.
